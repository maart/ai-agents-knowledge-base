# AI Agents Knowledge Base

Этот проект собирает и структурирует текстовую информацию о том, что такое AI в контексте агентских моделей: как устроены агенты, какие архитектуры используются, какие инструменты и стандарты вокруг них появились, и как из этого складывается новая прикладная экосистема.

## Навигация

Эта страница — главная точка входа. Для последовательного чтения начинай со [Study guide](study-guide.md); внутри разделов верхняя ссылка ведет к ближайшей полезной странице маршрута.

## С чего читать

1. [Study guide](study-guide.md) — основной маршрут изучения: overview -> concepts -> landscape -> sources -> drafts.
2. [Карта темы](00-map.md) — быстрый обзор области.
3. [Текущий синтез](notes/current-synthesis.md) — короткое резюме текущего понимания.
4. [Глоссарий](01-glossary.md) — рабочие определения ключевых терминов.
5. [Roadmap](roadmap.md) — план дальнейшего развития проекта, не основной маршрут чтения.

## Разделы

- [Concepts](notes/concepts/README.md) — базовые понятия и различия.
- [Landscape](landscape/README.md) — экосистема фреймворков, стандартов, продуктов и runtime-паттернов.
- [Sources](sources/README.md) — карточки papers, docs, essays, product pages и benchmarks.
- [Drafts](drafts/README.md) — связные тексты, собранные из заметок и источников.

## Рабочий принцип

Сначала строится не финальный текст, а исследовательская база:

- карта понятий;
- карточки источников;
- короткие заметки по темам;
- периодический синтез на 1-2 страницы;
- отдельный обзор ландшафта инструментов и продуктов.

Каждая заметка должна быть либо привязана к источнику, либо явно помечена как собственный синтез.
