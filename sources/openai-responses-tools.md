## Main idea

The Responses API makes tool use part of the model interaction surface. It supports custom function calling and built-in tools such as web search, file search, code interpreter, computer use and remote MCP servers.

## Relevance to AI agents

- Shows how tool use is becoming a standard API-level primitive.
- Important for understanding the difference between "model answers" and "model acts through tools".
- Connects agentic systems to MCP, computer use and long-running workflows.
